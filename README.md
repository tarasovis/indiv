# Типовой проект на Java

## Задача
Дано множество точек на плоскости и множество кругов. Среди всех треугольников,
вершины которых выбираются из множества точек, нужно найти такой треугольник,
что:

1. снаружи него находится максимальное количество кругов;
2. среди таких треугольников его периметр максимален.

Под словом **снаружи** понимается, что у треугольника и круга нет общих точек.

## Что реализовано
- меню выбора способа задания исходных данных;
- ввод данных с клавиатуры;
- загрузка данных из произвольного файла;
- загрузка из заранее подготовленных файлов `example_1.txt` и `example_2.txt`;
- визуальный ввод точек и кругов мышью;
- плавная отрисовка круга при движении мыши после выбора центра;
- сохранение текущих данных в файл;
- поиск лучшего треугольника;
- вывод результата на координатную плоскость и в строку состояния.

## Формат файла
```text
POINTS <количество_точек>
x1 y1
x2 y2
...
CIRCLES <количество_кругов>
cx1 cy1 r1
cx2 cy2 r2
...
```

## Управление
- **Данные → Ввод с клавиатуры** — текстовый ввод;
- **Данные → Загрузить пример 1/2** — загрузка готовых наборов;
- **Файл → Открыть...** — чтение из произвольного файла;
- **Файл → Сохранить...** — сохранение в файл;
- **Инструмент → Точка** — добавление точки кликом мыши;
- **Инструмент → Круг** — первый клик задает центр, движение мыши меняет радиус,
  второй клик фиксирует круг;
- **Действия → Найти лучший треугольник** — запуск алгоритма.

## Компиляция и запуск
Из папки `src`:
```bash
javac *.java
java Main
```

## Структура проекта
- `Main.java` — запуск программы;
- `MainFrame.java` — главное окно и меню;
- `DrawingPanel.java` — координатная плоскость и ввод мышью;
- `GeometryUtils.java` — геометрические вычисления;
- `TriangleSearcher.java` — перебор треугольников и выбор лучшего;
- `DataFileManager.java`, `TextDataParser.java` — чтение и запись данных;
- остальные классы — модели предметной области.

## Примечание
В проекте используются отдельные небольшие методы с понятными именами;
методы написаны так, чтобы не превышать ограничение в 25 строк кода.
