# Типовой проект на Java Swing

## Задача
Дано множество точек и множество кругов на плоскости.
Нужно перебрать все треугольники по точкам и выбрать такой, для которого:
1. количество кругов строго снаружи треугольника максимально;
2. среди них периметр треугольника максимален.

Круг считается снаружи только если у круга и треугольника нет общих точек.
Касание тоже считается пересечением.

## Интерфейс
- Меню **File**: `Open`, `Save`, `Example 1`, `Example 2`, `Exit`.
- На панели меню отдельные пункты: `Keyboard input`, `Clear`, `Find triangle`.
- ЛКМ добавляет точку.
- ПКМ: первый клик задает центр круга, движение мыши плавно меняет радиус,
  второй клик фиксирует круг.
- Результат поиска выводится через `JOptionPane` и рисуется на панели.

## Формат файла
```text
N
x1 y1
...
xN yN
M
cx1 cy1 r1
...
cxM cyM rM
```

## Ввод с клавиатуры
Каждая строка:
- `x y` — точка;
- `x y r` — круг.

## Ограничения реализации
- Используются только обычные массивы (`PlanePoint[]`, `PlaneCircle[]`).
- `List`, `ArrayList` и другие коллекции не используются.
- Методы разбиты на короткие блоки (до 25 строк).

## Запуск
```bash
cd src
javac *.java
java Main
```
